# Fuzzum Engine

## 2D and 3D cross-platform game engine

**Fuzzum Engine is a cross-platform
game engine to create 2D and 3D games.** 
Games can be exported with one click to a number of platforms, including the major desktop platforms (Windows), mobile platforms (Android).

## Free, open source and community-driven

Fuzzum Engine is completely free and open source under the MIT license

## Getting the engine

### Binary downloads

Official binaries for the Fuzzum Engine can be found in Releases.

### Compiling from source

1 Compile BuildTool(Run "./Engine/Source/Programs/FuzzumBuildTool/FuzzumBuildTool.sln" and compile)
2 Run "./Engine/Build.bat" to generate ninja build script
3 Run "./Build.bat" to compile engine
4 Launch Engine "./Engine/Binaries/Engine.exe"
